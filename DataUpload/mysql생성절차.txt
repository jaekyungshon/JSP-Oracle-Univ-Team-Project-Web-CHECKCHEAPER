[cmd로 mysql 접속]
1. mysql Server 다운 위치로 루트 변경 : (제 경로)C:\Program Files\MySQL\MySQL Server 8.0\bin
2. mysql -u root -p 입력
3. 설정한 관리자 비밀번호 입력.
4. 접속됨.
5. show DATABASES; (db 보기)
6. use 사용할db;
6-1. 초기에 생성해야함 -> create database 사용할db이름;
7. select 등등 사용 가능.
8. quit (종료)
